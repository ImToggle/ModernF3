# ModernF3

![Minimal Powered by OneConfig](https://polyfrost.org/img/cozy-minimal_vector.svg)

Backport F3 features from modern versions.

## Features

- Chunk borders
- Copy location as /tp
- Debug messages
- KeyBinding fixes
- Render distance control
- Singleplayer pause screen
